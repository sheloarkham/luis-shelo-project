# 🤖 Jack - Tu Amigo Imaginario Inteligente

Jack es un chatbot inteligente con **memoria infinita** que nunca olvida nada de lo que hablas con él. Puedes configurarlo con diferentes personalidades o darle tu propia filosofía.

## ✨ Características

- **🧠 Memoria Infinita**: Guarda TODA la conversación usando IndexedDB (sin límites)
- **🎭 Personalidades Configurables**: 
  - Mentor Sabio
  - Filósofo Estoico
  - Amigo Cercano
  - Coach Motivacional
  - Terapeuta Reflexivo
  - Personalizado (tú defines cómo es)
- **💾 Persistencia Local**: Todo se guarda en tu navegador
- **📥 Exportación**: Descarga backup de todas tus conversaciones
- **🆓 100% Gratis**: Usa Google Gemini API (sin tarjeta de crédito)

## 🚀 Cómo Empezar

### 1. Obtener API Key (Gratis)

1. Ve a [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Inicia sesión con tu cuenta de Google
3. Haz clic en **"Create API Key"**
4. Copia la key generada

### 2. Configurar Jack

1. Ve a la página **Jack** en el navbar
2. Haz clic en **"⚙️ Configurar"**
3. Pega tu API Key
4. Elige una personalidad (o crea la tuya)
5. ¡Listo! Empieza a hablar

## 🎭 Personalidades Disponibles

### 🎓 Mentor Sabio
Guía y enseña a través de preguntas reflexivas. Perfecto para desarrollo personal y aprendizaje.

### 🏛️ Filósofo Estoico
Basado en Marco Aurelio, Epicteto y Séneca. Te ayuda a ver las cosas desde la perspectiva del control interno.

### 🤝 Amigo Cercano
Escucha sin juzgar, da consejos honestos. Como hablar con un amigo de confianza.

### 💪 Coach Motivacional
Directo, positivo, enfocado en acción y resultados. Te impulsa a superar obstáculos.

### 🧘 Terapeuta Reflexivo
Hace preguntas poderosas para explorar emociones y patrones. Crea un espacio seguro.

### ⚙️ Personalizado
Define tu propia personalidad. Ejemplos:
- "Eres un experto en programación que explica con analogías simples"
- "Eres un gurú de fitness motivacional y estricto"
- "Eres un consejero financiero conservador"

## 💾 Gestión de Memoria

### Ver estadísticas
En el header verás: `💾 X mensajes guardados`

### Exportar conversación
Haz clic en **📥** para descargar un JSON con toda tu historia.

### Borrar memoria
Haz clic en **🗑️** para limpiar toda la conversación (confirmación requerida).

## 🔒 Privacidad

- ✅ Toda la memoria se guarda **localmente** en tu navegador
- ✅ No se envía a ningún servidor (excepto a Gemini para generar respuestas)
- ✅ Puedes borrar todo en cualquier momento
- ✅ La API Key se guarda en localStorage

## 🆓 Límites Gratuitos

**Google Gemini API Gratis:**
- ✅ 60 requests por minuto
- ✅ Sin límite de mensajes diarios
- ✅ Sin necesidad de tarjeta de crédito
- ✅ Modelo: Gemini 1.5 Flash (rápido y potente)

## 🛠️ Tecnologías Usadas

- **Frontend**: React + Vite
- **API**: Google Gemini 1.5 Flash
- **Almacenamiento**: IndexedDB (memoria infinita)
- **Estilos**: CSS custom con gradientes

## 💡 Tips de Uso

1. **Dale contexto inicial**: "Eres mi mentor de productividad..."
2. **Sé específico**: Jack tiene TODA tu historia, úsala
3. **Experimenta personalidades**: Cambia según tu necesidad
4. **Exporta regularmente**: Backup de tus reflexiones importantes
5. **Respeta el rate limit**: 60 req/min (más que suficiente)
